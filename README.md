# 烧写测试一体上位机

----------
### 一、介绍
  stm32系列产品，可通过上位机实现一键烧写配置和测试功能，并可将配置参数导入，测试结果导出。 有调试页面，集成串口和网口的调试
### 二、功能点
1. 固件烧写（hex）
  - 进度条
  - 烧写固件手动选择，自动烧写
2. 配置文件导入(config)
  - 显示配置
3. 测试结果导出（excel）
4. 串口/网口通讯
  - 波特率/ip地址与端口可配置，输入字符串与hex格式可切换 
  - 可添加和删除串口配置命令
  - 窗口可清空，log可保存（txt）
  - 可定时发送测试数据
  - 有帧解释器与编辑器
  - 固件升级功能 --升级固件手动选择
  - ascii 码表
  - crc 计算工具
